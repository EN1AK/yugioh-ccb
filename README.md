## yugioh-ccb

本项目为灯一把——游戏王版。

数据来源：ygopro,mycard.world

### 玩法

1、浏览器打开 111.229.8.218

2、选择题库（目前仅推荐热门怪兽卡，其他太难）

3、第二、五次猜测会给出提示


## 部署指南

   ```
   pip install -r requirements.txt
   python guess_card_game.py 
   ```

浏览器打开http://127.0.0.1:5000
